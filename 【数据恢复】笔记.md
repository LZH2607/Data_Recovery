## 存储格式

大头位序（Big-endian）：高位在左，低位在右
	文件系统：HFS

小头位序（Little-endian）：低位在左，高位在右
	文件系统：FAT32、NTFS、exFAT



## WinHex

![](图片/WinHex界面.png)

字节（Byte，B）

1行 = 16B
1扇区 = 32行 = 512B
1KB = 2扇区 = 1024B



## 文件系统

文件系统：
	FAT32（File Allocation Table）
	NTFS（New Technology File System）
	exFAT（Extended File Allocation Table）



## FAT32

FAT32文件系统结构：
	DBR：系统引导扇区
	保留区
	FAT1
	FAT2
	数据区
